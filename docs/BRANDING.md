# 브랜딩 가이드

이 앱의 정체성, 이름, 목소리, 시각적 일관성을 정의하는 문서.
UI 디자인 디테일(색상 토큰, 컴포넌트 사양 등)은 `DESIGN.md` 참고.

## 1. 프로젝트 이름

**확정 이름**: **Claudette** (2026-05 결정)

> Claude 의 프랑스 지소사 (diminutive). "작은 Claude" = 사용자 옆에서 사용량·크레딧을 조용히 알려주는 작은 도우미 컨셉.

### 결정 이유

1. **상표 안전성** — Claudette 는 독립된 프랑스 이름이라 Anthropic 의 Claude 상표와 별개. "Claude" 를 직접 포함한 후보 (Claudely, Claudemeter 등) 대비 법적 리스크 낮음.
2. **즉시 연관성** — 사용자가 보면 즉시 Claude 가 떠오름. 검색·App Store 노출에도 유리.
3. **기능 확장성** — 구독 한도 + 크레딧 (`claude-p`) + 향후 추가될 메트릭까지 "내 Claude 도우미" 컨셉으로 자연스럽게 묶임.
4. **친근한 톤** — 본 문서 §4 의 페르소나 ("친근한, 사려깊은") 와 자연스럽게 맞물림.

### 검토된 후보 (참고 기록)

| 이름 | 느낌 | 탈락 사유 |
|------|------|------|
| Pulse | 실시간 모니터링, 맥박 | 동명 앱 다수 (Pulse Secure 등), 도메인 확보 어려움 |
| Gauge | 계기판, 직관적 | 무미건조, 캐릭터화 어려움 |
| Tally | 카운트, 친근 | 회계 SW Tally 와 동명 |
| Claudely | Claude + ly | Claude 직접 포함 → 상표 리스크 |
| MaxBar | Claude Max 플랜 | 크레딧으로 스코프 확장 시 무의미 |
| Mete · Brimm · Stash | 추상 계량 메타포 | 의미는 좋으나 Claude 연관성 약함 |
| Throttle · 사용량요정 | — | 톤 부적합 / 글로벌화 어려움 |

> "Claude 한도를 신경 쓰지 않고 일에 집중할 수 있도록."

## 3. 타깃 페르소나

**메인**: Claude Max 플랜을 매일 쓰는 macOS 사용자. 개발자/디자이너/리서처가 다수. 작업 흐름이 끊기는 것을 매우 싫어함.

핵심 인사이트: 이들은 "한도 임박했나?" 가 궁금해서 매번 claude.ai 들어가는 게 마이크로 컨텍스트 스위치라고 느낌. Claudette 는 그 마찰을 0 으로 줄여주는 가벼운 도우미.

---

## 4. 브랜드 퍼스널리티

5개 형용사로 정의:

1. **간결한 (Concise)** — 군더더기 없음, 필요한 정보만
2. **차분한 (Calm)** — 자주 떠 있는 도구라 시끄럽지 않아야 함
3. **신뢰감 있는 (Reliable)** — 숫자가 정확하고 빠르게 업데이트됨
4. **친근한 (Friendly)** — 차갑지 않음, 가볍게 농담도 함
5. **사려깊은 (Considerate)** — 사용자 흐름을 방해하지 않음 (알림 빈도, snooze 등)

**아닌 것 (anti-personality)**:
- 화려한 (X)
- 시끄러운 (X)
- 기업스러운 (X)
- 잘난체하는 (X)

## 5. 보이스 & 톤

### 5.1 일반적인 보이스

평소엔 **친근한 동료** 톤. 존댓말과 반말 사이의 가벼운 존중. 영문은 자연스러운 일상 영어.

### 5.2 상황별 톤

| 상황 | 톤 | 예시 |
|------|----|-----|
| 정상 상태 표시 | 무덤덤, 사실 중심 | "현재 세션 52%" / "8% used this week" |
| 한도 임박 (80%+) | 침착하게 알림, 호들갑 X | "한도 80% 도달. 페이스 조절 추천." / "Pacing toward limit" |
| 한도 도달 (100%) | 짧고 명확 | "한도 도달. 5시간 후 재설정." / "Limit reached." |
| 에러 | 사용자 탓 안 함, 다음 행동 명확 | "claude.ai 응답이 없어요. 잠시 후 자동 재시도합니다." |
| 로그인 안내 | 가볍게, 친절히 | "한 번만 로그인하면 끝나요." |
| 빈 상태 | 살짝 유머 | "Claude 안 쓰고 계시네요. 평화롭네요 ☕️" |

### 5.3 절대 쓰지 않을 것

- ❌ "Oops!" / "헉" / "당신은 한도를 초과했습니다!" → 호들갑
- ❌ "AI-powered productivity dashboard" → 마케팅 BS
- ❌ "지금 당장 업그레이드하세요" → 영업
- ❌ "오류가 발생했습니다. 관리자에게 문의하세요." → 책임 회피

## 6. 로고 컨셉

### 6.1 컨셉

심볼은 **계기판(gauge) + 맥박(pulse)** 의 조합. 한눈에 "측정 도구" 임을 알 수 있고, 가볍게 살아 움직이는 느낌.

### 6.2 가이드

- 단순한 기하학적 형태, 두꺼운 라인 (메뉴바 아이콘으로 작아져도 인식 가능해야 함)
- 색은 단색 또는 2색 (기본 + 액센트)
- SF Symbol `gauge.medium` 을 베이스로 한 후 커스터마이즈 가능
- 메뉴바 표시 시: 시스템 아이콘 모양(@templateImage)으로 만들어 다크/라이트 자동 적응

### 6.3 메뉴바 아이콘 변형

상태별로 색상만 다른 동일 심볼:
- 🟢 normal (`< warningThresholdPercent`)
- 🟡 warning (`>= warningThresholdPercent`)
- 🔴 danger (`>= dangerThresholdPercent`)
- ⚪️ neutral (미로그인 / 데이터 없음)

## 8. 타이포그래피 (브랜드용)

앱 UI 폰트는 시스템 (SF Pro). 마케팅 페이지나 로고 워드마크용으로 사용할 폰트만 여기 정의:

- **워드마크**: SF Pro Rounded Bold (시스템 폰트 활용, 라이선스 무료)
- **본문 (마케팅)**: SF Pro
- **고정폭 (코드 표시)**: SF Mono

브랜드 차원에서 추가 폰트 구매하지 않음 (라이선스 비용 절감 + macOS 네이티브 일관성).

## 9. 메시지 라이브러리

자주 등장하는 문장은 통일된 표현으로:

| 상황 | 한국어 | 영어 |
|------|--------|------|
| 첫 화면 메인 CTA | Claude에 로그인 | Sign in to Claude |
| 로그인 보조 텍스트 | 한 번만 로그인하면 됩니다 | One-time setup |
| 새로고침 버튼 | 새로고침 | Refresh |
| 설정 진입 | 설정 | Settings |
| 로그아웃 | 로그아웃 | Sign out |
| 종료 | 종료 | Quit |
| 곧 재설정 | 곧 재설정 | Resetting soon |
| N분 후 재설정 | {N}분 후 재설정 | Resets in {N}m |
| 한도 도달 알림 | 한도 {N}% 도달 | {N}% reached |
| 페이스 알림 | 약 {N}분 후 한도 도달 예상 | Limit in ~{N}m at this pace |

새 문구 추가할 때 이 표 업데이트.
